# Justin's Laboratory

A collection of browser-based tools and utilities, hosted under [lab.justinbornais.ca](https://lab.justinbornais.ca). All tools are **100% frontend-only:** no data you insert or upload ever leaves your browser.

## Tools

| Tool | Description | GitHub |
|------|-------------|--------|
| [PDF Viewer](https://pdf-viewer.lab.justinbornais.ca) | A lightweight PDF viewer with horizontal scrolling, ideal for musicians reading sheet music. | [justinbornais/pdf-viewer](https://github.com/justinbornais/pdf-viewer) |
| [PDF Cropper](https://pdf-cropper.lab.justinbornais.ca) | An advanced PDF cropping tool that can split pages mid-page, separate into multiple documents, and merge pages together. | [justinbornais/pdf-cropper](https://github.com/justinbornais/pdf-cropper) |
| [SQLite Viewer](https://sqlite.lab.justinbornais.ca) | A simple SQLite viewer and playground. Upload a database to browse its contents and run queries interactively. | [justinbornais/sqlite-viewer](https://github.com/justinbornais/sqlite-viewer) |
| [Slideshow Generator](https://slideshow.lab.justinbornais.ca) | Create slideshows from uploaded images with custom text, transitions, and music. Exports directly to MP4. | [justinbornais/slideshow-generator](https://github.com/justinbornais/slideshow-generator) |
| [Cardify](https://cardify.lab.justinbornais.ca) | Batch-print cards on a sheet of paper. Automatically fits the maximum number of cards per page, with support for double-sided printing. | [justinbornais/cardify](https://github.com/justinbornais/cardify) |
| [Investment Calculator](https://investment.lab.justinbornais.ca) | A simple investment calculator for projecting net worth over time and determining the required contributions to reach a target. | [justinbornais/investment-calc](https://github.com/justinbornais/investment-calc) |
| [Spotweld Plotter](https://stp.lab.justinbornais.ca) | Upload a .stp or CAD file to plot spotwelds on a 3D model. Automatically traces paths between spots and exports location data to JSON. | [justinbornais/stp-spotweld-exporter](https://github.com/justinbornais/stp-spotweld-exporter) |
